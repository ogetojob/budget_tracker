# Expense Tracker  

A simple Jupyter Notebook project for tracking and analyzing personal expenses.  

## 📌 Features  
- Record daily expenses  
- Categorize spending (e.g., food, transport, bills)  
- Summarize total expenses  
- Basic charts/visualizations
